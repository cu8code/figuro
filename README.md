## 📖 How to Get Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/figuro-monorepo.git
   cd figuro-monorepo
   ```

2. Install dependencies for all projects:
   ```bash
   pnpm install
   ```

3. Run individual projects:
   - **Site (Frontend + Backend)**:
     ```bash
     cd site
     pnpm run dev
     ```
   - **Choco**:
     ```bash
     cd choco
     python main.py
     ```
